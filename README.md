# Ittan

ittan（一旦）は`schema.rb`からとりあえずseed情報を生成するGemです。

※ seed_fuを想定しております

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ittan'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ittan

## Usage

```
$ bundle exec ittan
```
