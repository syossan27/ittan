# Ittan

Ittan (一旦) is a Gem that generates seed file from schema.rb for the time being.

※ It is assuming seed_fu.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ittan'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ittan

## Usage

```
$ bundle exec ittan
```
